# SuperStore Sales Analysis

## Overview

This project performs a comprehensive analysis of retail sales data from a SuperStore dataset. The objective is to uncover business insights, identify sales trends, evaluate profitability, and support data-driven decision-making through visualization and analysis.

The project was completed as part of a Real-World Data Analysis Project using Python and data analytics techniques.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## Dataset Information

The SuperStore dataset contains:

* Customer Information
* Product Information
* Sales Data
* Profit Data
* Shipping Details
* Regional Information

### Dataset Summary

* Total Records: 9,993
* Total Features: 25
